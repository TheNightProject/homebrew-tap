# Homebrew Tap

Official Homebrew tap for [The Night Project](https://github.com/TheNightProject) apps.

## Install

```bash
brew tap TheNightProject/tap
brew install --cask claude-lens
```

## Available Casks

| Cask | Description |
|------|-------------|
| `claude-lens` | Local observability dashboard for Claude & AI developer tooling |

## Update

```bash
brew upgrade --cask claude-lens
```

## Uninstall

```bash
brew uninstall --cask claude-lens
brew untap TheNightProject/tap
```
